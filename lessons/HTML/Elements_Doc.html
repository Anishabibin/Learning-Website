HTML Elements

What is an HTML Element? It includes a start tag, some content, and an end tag.

<tagname>Content goes here...</tagname>

Examples:

Start Tag Content End Tag
<h1> My First Heading </h1>
<p> My first paragraph. </p>
<br> None None
<html> <body> <h1>My First Heading </h1> <p>This is my paragraph without closing the tag... </p> </body> </html>
